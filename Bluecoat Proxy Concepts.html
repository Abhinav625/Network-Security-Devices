Checking logs on Bluecoat Proxy

https://<proxyhost>:8082/eventlog/fetch=0xFFFFFFFF     All Events

https://<proxyhost>:8082/eventlog/fetch=0x00000002     Errors

https://<proxyhost>:8082/eventlog/fetch=0x00000003     Priority 1 Events

https://<proxyhost>:8082/eventlog/fetch=0x00001000     SSL

https://<proxyhost>:8082/eventlog/fetch=0x00008000     Configuration Errors

https://<proxyhost>:8082/eventlog/fetch=0x02000000     Configuration changes

https://<proxyhost>:8082/eventlog/fetch=0x10000000     Download and Uploads (logs, BCWF, ...)
